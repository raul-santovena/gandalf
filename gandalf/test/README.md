# Test

This module contains the test methods for evaluating GANDALF models. The available methods are:

 - `tsne_comparison`: comparison of spectra and latent space using the tsne algorithm (``tsne.py``).

 - `conditional_parameters_analysis`: analyze the result of predict the conditional parameters from the spectra vs from the latent space (``cond_params.py``).

 - `no_conditional_parameters_analysis`: analyze the result of predict no conditional parameters from the spectra vs from the latent space (``cond_params.py``).
