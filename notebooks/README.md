# Notebooks 📓

The following notebooks are available to test GANDALF and learn how it works.

- ``1-gandalf-train.ipynb``: run a complete training process with GandalfTrain using the sample data. Configuration available in ``config.yaml``. Must be run before the next notebooks.

- ``2-tsne.ipynb``: generate a t-SNE from a Gandalf experiment.

- ``3-cond-prediction.ipynb``: run the conditional parameter analysis.

- ``4-bokeh-gandalf-server.ipynb``: creates and starts a Bokeh server as if using the `bokeh serve` command.
